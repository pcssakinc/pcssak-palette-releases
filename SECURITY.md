# 보안 제보 / Security

업데이트 서명 우회, 권한 검사 우회, 임의 파일 접근, 이미지 처리 취약점처럼 악용 가능한 문제는 공개 Issue에 상세 재현 코드를 올리지 말고 `pcssakinc@gmail.com`으로 보내주세요.

다음 정보를 포함하면 확인에 도움이 됩니다.

- PCssak Palette 버전과 Windows 버전
- 문제의 영향과 재현 조건
- 필요한 최소 재현 자료
- 공개해도 되는 연락 방법

비밀번호, 결제 정보, 주민등록번호, 고객 정보 또는 회사의 비공개 원본 디자인은 보내지 마세요. 접수 사실을 확인한 뒤 영향과 수정 방법을 검토하고, 공개 시점은 제보자와 협의하겠습니다.

For an exploitable vulnerability, please email `pcssakinc@gmail.com` instead of posting public exploit details. Include the app and Windows versions, impact, minimal reproduction steps, and a safe contact method. Do not send passwords, payment data, personal identifiers, customer data, or confidential original designs.
