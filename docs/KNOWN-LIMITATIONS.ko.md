# 알려진 한계

[English](KNOWN-LIMITATIONS.md)

## 현재 릴리스

- GitHub 배포본은 1.0 이전의 무료 Early Access이며 결제와 Pro 권한이 없습니다.
- 1.0 전에는 기능·화면·번역·파일 동작이 바뀔 수 있습니다.
- 발견되지 않은 오류와 호환성 문제가 남아 있을 수 있습니다.
- 1인 개발자가 지원하므로 응답이나 수정 기한을 보장하지 않습니다.

## 설치 파일과 Windows

- 최초 NSIS 설치 파일에는 Authenticode 서명이 없어 알 수 없는 게시자나 SmartScreen이 나타날
  수 있습니다. 공식 PCSSAK 경로만 사용하고 `SHA256SUMS.txt`를 확인하세요.
- Windows 11 Home/Pro x64는 공식 검증, Windows 10 22H2 x64·x86은 베타 시험 지원입니다.
- ARM64 네이티브, Windows S 모드, macOS, Linux는 배포하지 않습니다.
- Windows 10 Home/Pro는 Microsoft 일반 지원이 끝났으며 앱 지원이 운영체제 보안 업데이트를
  대신할 수 없습니다.

## 색 해석

- 색각이상 보기는 근사 시뮬레이션이며 개인 측정이나 의료 진단이 아닙니다.
- 색 거리 위험 기준은 PCSSAK 검토 휴리스틱이며 WCAG나 임상 기준이 아닙니다.
- 대비 결과는 표시한 색 조합에만 적용되며 제품 전체 접근성을 인증하지 않습니다.
- 아름다움, 문화, 브랜드 적합성이나 전문적 성공을 판단하지 않습니다.
- 모니터 색역·보정·조명·색상 관리·인쇄에 따라 실제 모습이 달라질 수 있습니다.

## 파일과 저장

- PNG·JPEG·WebP·GIF·BMP를 지원하고 HEIC·AVIF·PSD·RAW·SVG는 추출하지 않습니다.
- 40 MiB 또는 4,000만 픽셀을 넘는 파일은 거부하며 손상된 지원 형식도 실패할 수 있습니다.
- 라이브러리는 로컬이며 클라우드 동기화와 서버 복구가 없습니다. 초기화·삭제·기기 이동 전 JSON
  백업을 내보내고 중요한 결과물은 별도로 보관하세요.
- Free는 최대 10개를 저장합니다. 향후 Pro 저장·납품 기능은 아직 구매할 수 없습니다.

## 저장소

- 이 공개 저장소는 독점 바이너리와 문서를 배포하며 비공개 애플리케이션 소스의 오픈소스
  공개가 아닙니다.
- 제3자 구성요소는 각각의 오픈소스 라이선스와 고지문을 따릅니다.
